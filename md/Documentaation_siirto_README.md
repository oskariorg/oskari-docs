```
Kaikki vanhat dokumentaatiot on aluksi vain numeroitu, niiden siirto on helpompi taulukoida.
Ensimmäiseen taulukkoon on merkitty numeroinin perusteella mikä tiedosto kyseessä, onko se siirretty
ja mistä se löytyy uudesta dokumentaatiosta. Toiseen taulukkoon taas on merkkitty mitä kaikkea on
vanhasta dokumentaatiosta siirettyy tiettyyn kohtaan uutta. Taulukoita on melkein järkevintä ajatella
kuin kaksinkertaista kirjanpitoa ja dokumentin siirron jälkeen siitä kuuluu löytyä merkintä molemmista 
taulukoista.
```
